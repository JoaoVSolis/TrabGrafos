# TrabGrafos
O Trabalho irá apresentar uma aplicação prática de desenvolvimento de um sistema de análise de grafos. A aplicabilidade possui o intuito de implementar e testar os principais algoritmos clássicos de grafos estudados na disciplina: Travessia em Profundidade (DFS), Travessia em Amplitude (BFS), Dijkstra, Prim e Kruskal.
	A elaboração da aplicabilidade permite criar, manipular e analisar um grafo de forma interativa pelo console. De começo, através do tamanho do grafo fornecido pelo usuário, um menu principal surge onde pode realizar várias operações, como: adicionar arestas ao grafo informando o vértice de origem, o vértice de destino e o peso dessa ligação; também pode remover uma aresta já existente informando origem e destino. Além disso, o software permite representar o grafo de três maneiras diferentes: como matriz de adjacência, lista de arestas ou lista de adjacência, facilitando a visualização da sua estrutura.
	O programa também inclui um menu exclusivo para execução dos algoritmos de grafos citados anteriormente. Em cada caso, o usuário informa os vértices necessários e o algoritmo é executado diretamente no grafo criado. No geral, esse software funciona como um ambiente para construção, edição, visualização e execução de algoritmos sobre grafos.
